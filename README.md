# GYMX - Fat Loss Application

Welcome to the GYMX app! This application is built with React and Vite to provide a premium, fast experience.

## 🚀 How to Open the App

To view and run the application locally, please follow these steps:

1.  **Open your terminal** in the project directory.
2.  **Install dependencies** (if you haven't already):
    ```bash
    npm install
    ```
3.  **Start the development server**:
    ```bash
    npm run dev
    ```
    *Note: `npm start` also works as an alias.*

4.  **View the app**:
    Once the server starts, it should automatically open your browser to `http://localhost:5173/`.
    If it doesn't open automatically, you can manually go to [http://localhost:5173/](http://localhost:5173/) in your web browser.

> [!IMPORTANT]
> Do NOT try to open the `index.html` file directly in your browser. The application requires a running development server (Vite) to handle the React components and styling correctly.

## 🛠️ Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Vanilla CSS
- **Icons**: Lucide React
- **Routing**: React Router DOM

## 📝 Project Structure

- `src/components`: Reusable UI components.
- `src/pages`: Main application views.
- `src/data`: Static data for plans and stats.
- `src/context`: React context providers for state management.
